Итератор — это объект, который осуществляет доступ к элементам последовательности. 
Для того чтобы объект являлся итератором в нем должно быть реализовано два метода:

__iter__() - метод, который возвращает сам объект.
next() - метод, который возвращает следующее значение итератора.

Генератор(генератор-итератор, генераторное выражение) - это функция возвращающая следующее значение итератора.
При этом итератор не создается полностью(не создается вся итерируемая последовательность),
а лишь возвращается(генерируется) одно значение для каждой итерации.
Генератор создается с использованием оператора yield.
yield работает как return, с одной разницей, что между вызовами функций, все состояния и данные будут при следующем вызове функции такими, какими они были на момент предыдущего исполнения yield. 
При выходе из функции не по оператору yield автоматически генерируется StopIteration.
